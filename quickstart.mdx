## Setup your development

Learn how to update your docs locally and deploy them to the public.

<AccordionGroup>
  <Accordion title="Mac:">
    1. Run `yarn install` or `npm install` to install dependencies
    2. Run `brew install gnupg` 
    3. Run `brew install dopplerhq/cli/doppler`
    4. Check doppler is running with `doppler --version`
    5. Create a new account on Doppler if you don't have one
    6. Ask to get added to Doppler workspace
    7. Login to Doppler `doppler login`
    8. cd into the /fuxam-web home directory and run `doppler setup`
    9. Select `dev_vars` and click enter
  </Accordion>
  <Accordion title="Windows:">
    1. Download scoop if you don't have it already by running `npm install scoop`
    2. Run `scoop bucket add doppler https://github.com/DopplerHQ/scoop-doppler.git `
    3. Run `scoop install doppler`
    4. Check doppler is running with `doppler --version`
    5. Create a new account on Doppler if you don't have one
    6. Ask to get added to Doppler workspace
    7. Login to Doppler in your terminal `doppler login`
    8. cd into the /fuxam-web home directory and run `doppler setup`
    9. Select your environment and click enter
  </Accordion>
</AccordionGroup>
