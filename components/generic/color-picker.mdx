### How it Works

This component is a color-tastic combo of a text input and a color picker. Click the button, and a popover appears with an input field and a color picker. Type in a new color code or pick a color, and it updates in real-time. Easy peasy!

### Props

- `color`: The initial color value (defaults to `#FF5675`)
- `onChange`: A function that gets called whenever the color changes, passing the new color value as an argument

### Example Usage

### Basic Example

```jsx
import ColorPicker from './ColorPicker';

function MyComponent() {
  const [color, setColor] = useState('#FF0000');

  return (
    <ColorPicker
      color={color}
      onChange={(newColor) => {
        setColor(newColor);
        console.log(`New color: ${newColor}`);
      }}
    />
  );
}

```

### Advanced Example with Custom Button Text

```jsx
import ColorPicker from './ColorPicker';

function MyComponent() {
  const [color, setColor] = useState('#00FF00');

  return (
    <ColorPicker
      color={color}
      onChange={(newColor) => {
        setColor(newColor);
        console.log(`New color: ${newColor}`);
      }}
    >
      <Button variant="outline">Change Color!</Button>
    </ColorPicker>
  );
}

```
